/////////////////
To install...

1. Enter to console:
git init

2. Enter to console:
git clone https://github.com/IgorEvtushenko/octarine.git

3. Enter to console:
cd octarine

4. Enter to console:
npm install

/////////////////
To run

1. To run the development version, enter to console:
npm run dev

Or

2. To run the production version, enter to console:
npm run prod
